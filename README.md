**Claude Learning Guide**
A collection of beginner-friendly guides for getting started with Claude Code and AI-powered development tools. Designed for non-technical users — designers, PMs, marketers, and anyone curious about coding with AI.

Who This Is For
These guides are written for people with little or no coding experience who want to start using AI tools like Claude Code in their day-to-day work. No prior terminal or programming knowledge is assumed.
How to Use
Open any of the .html files directly in your browser — no server or installation required. The Word document can be opened in Microsoft Word or Google Docs.
